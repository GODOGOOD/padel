# 🎾 Padel Coach App

A training planner for padel coaches.

## Features
- Lesson planning (I / A1 / A2 / B / E)
- Player management
- Checklist (KNLTB structure)
- Save / Load local storage
- PDF export (print to PDF)

## Run
Just open index.html in browser.

## Future upgrades
- Drag & drop court system
- Firebase login system
- Club dashboards
